# Wanderlust - Vacation Listing Application

A full-stack web application inspired by Airbnb that allows users to explore, create, and manage vacation listings. Users can register, log in, upload listing images, view locations on a map, and post reviews.

## 🚀 Features

* User Authentication and Authorization
* Create, Edit, and Delete Listings
* Upload Images using Cloudinary
* Interactive Maps Integration
* Review and Rating System
* Flash Messages and Form Validation
* Responsive UI
* Session and Cookie Management
* MVC Architecture
* Error Handling Middleware

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Ruturaj-Avaghade/Wanderlust-vacation-listing-app.git
```

### Navigate to the project directory

```bash
cd Wanderlust-vacation-listing-app
```

### Install dependencies

```bash
npm install
```

### Create a .env file

```env
ATLASDB_URL=
SECRET=
CLOUD_NAME=
CLOUD_API_KEY=
CLOUD_API_SECRET=
MAP_TOKEN=
```

### Start the application

```bash
node app.js
```

or

```bash
nodemon app.js
```

## 🏗️ Architecture

This project follows the MVC (Model-View-Controller) architecture:

* Models → Database schema and collections
* Views → EJS templates
* Controllers → Business logic
* Routes → Application endpoints
